Main Components

  • package.json - Extension Manifest
  • extension.ts - Entry Point
  • Contribution Points
  • Activation Events

package.json manifest

{
  ...
  "contributes": {
    "commands": [{
      "command": "myExtension.helloWorld",
      "title": "Hello World"
    }],
    "keybindings": [{
      "command": "myExtension.helloWorld",
      "key": "ctrl+shift+h",
      "mac": "cmd+shift+h",
      "when": "editorTextFocus"
    }]
  },
  "activationEvents": [
    "onLanguage:javascript",
  ]
}

Disposables & Subscriptions

  • Disposables manage extension resources
  • Used to clean up event listeners, commands, etc.
  • Added to extension context subscriptions

let disposable = vscode.commands.registerCommand(
  'extension.command', () => {
    // Command implementation
});

// Add to subscriptions
context.subscriptions.push(disposable);

Contribution Points

  • commands - Add commands to VS Code
  • menus - Add items to UI menus
  • keybindings - Add keyboard shortcuts
  • configuration - Add settings
  • views - Add custom views

Activation Events

  • onLanguage - When a file type is opened
  • onView - When a view is opened
  • onStartupFinished - When VS Code is ready
  • * - On VS Code startup (do not use)
  • many more...
