package.json manifest

{
  ...
  "contributes": {
    "commands": [{
      "command": "myExtension.helloWorld",
      "title": "Hello World"
    }],
    "keybindings": [{
      "command": "myExtension.helloWorld",
      "key": "ctrl+shift+h",
      "mac": "cmd+shift+h",
      "when": "editorTextFocus"
    }]
  },
  "activationEvents": [
    "onLanguage:javascript",
  ]
}

Subscriptions og Disposables

  • Registrer kommandoer, hendelseslyttere
  • Rydd opp ved avlasting

let disposable = vscode.commands.registerCommand(
  'extension.command', () => {
    // Command implementation
});

// Add to subscriptions
context.subscriptions.push(disposable);

Contribution Points

  • commands - Legg til kommandoer i VS Code
  • menus - Legg til elementer i menyer
  • keybindings - Legg til tastatursnarveier
  • configuration - Legg til innstillinger
  • views - Legg til egendefinerte visninger

Activation Events

  • onLanguage - Når en filtype åpnes
  • onView - Når en visning åpnes
  • onStartupFinished - Når VS Code er klar
  • * - Ved VS Code-oppstart (ikke bruk)
  • mange flere...
